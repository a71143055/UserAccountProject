📊 이메일 vs 클라우드 드라이브 기능 비교

분류 이메일 서비스 클라우드 드라이브
기본 화면 받은편지함, 보낸편지함, 임시보관함 내 드라이브, 공유 문서함, 최근 항목
정리 기능 라벨, 중요표시, 별표 폴더, 색상 태그, 즐겨찾기
검색 기능 제목, 내용, 보낸 사람 검색 파일명, 유형, 작성자 검색
작성 기능 새 메일 작성 및 첨부 새 폴더/문서/스프레드시트 생성
공유 기능 이메일 직접 전송, CC/BCC 링크 공유, 권한 설정
휴지통 삭제 메일 보관 삭제 파일 복구 가능
설정 서명, 필터, 전달 규칙 용량 확인, 권한 설정

📌 기능별 설명

  • 기본 화면: 이메일은 받은편지함 중심, 드라이브는 내 드라이브 중심입니다.
  • 정리 기능: Gmail은 라벨, Drive는 폴더/색상 태그로 정리합니다.
  • 검색 기능: 둘 다 고급 조건 검색을 지원합니다.
  • 작성 기능: 이메일은 메일 작성, 드라이브는 문서/시트 생성입니다.
  • 공유 기능: 이메일은 직접 전송, 드라이브는 링크 공유입니다.
  • 휴지통: 삭제 후 복구 가능성이 있는 임시 공간입니다.
  • 설정: 이메일은 서명/필터, 드라이브는 저장공간/권한 관리 중심입니다.
